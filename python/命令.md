1.安装: 在 https://www.python.org/downloads 下载对应平台的安装包
```
tar -zxvf Python-3.6.1.tgz

cd Python-3.6.1

./configure

make && make install
```

2.查看版本
```
python3 --version
```

3.进入python，终端输入：
```
python
```

4.退出python程序<code>quit</code> 或者<code>exit</code> ， 也可以使用快捷键Ctrl+D
```
>>> quit()
```

5.终端执行python文件
```
python3 hello.py
```
