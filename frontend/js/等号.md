`!==` 与 `===` 判断左右相等时不仅值相等，还回校验俩边类型是否一致

```
2 === '2' // false

2 !== '2' // true

let param1,param2 = '2',2

2 === param1 // false

2 !== param1 // true

2 === param2 // true

2 !== param2 // false

```

同理 `==` 与 `!=` 判断左右相等时判断值相等，不强校验类型

```
2 == '2' // true

2 != '2' // false


let param1,param2 = '2',2

2 == param1 // true

2 != param1 // false

2 == param2 // true

2 != param2 // false
```
